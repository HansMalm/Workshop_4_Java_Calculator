# Workshop Java CalculatorApp

### Pseudocode

```text
START
end = No
number1 = 0
number2 = 0
select = 0
PRINT "welcome"
WHILE end == No DO
  Print "instruction"
  number1 = INPUT
  number2 = INPUT
  select = INPUT
  SWITCH = select
    case1 addition
    case2 subtraction
    case3 multiplication
    case4 division
  PRINT "new calculation?"
  end = input
END
```
